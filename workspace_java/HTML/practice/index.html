1. 프로젝트 개발 업무 분담

(1) 바둑판에 사용할 html태그 레이아웃 구성

- 테이블 태그가 아닌 div태그로 구성

- 바둑돌 둘 곳을 식별할 수 있도록 해야함

- 태그를 구성하는데 반복적인 작업을 피하기 위해서 이중반복문을 이용한 태그 생성

(2) 바둑돌이 놓이는 클릭이벤트 구현

- 클릭시 순서에 따라 입력되는 그림을 다르게 하기 - 흑백 구분

- 이미 둔 곳에는 돌이 추가되지 않도록 하기 - 클릭이벤트 종료

- 바둑돌 이미지 태그 삽입시 돌의 위치 맞추기

(3) ajax를 이용해서 클릭이벤트 발생시 필요한 파라미터 가공

- 세션에 바둑돌에 대한 위치 저장

- 입력받은 정보를 기반으로 오목여부 확인하기

- 기존에 작성했던 오목 알고리즘을 프로젝트에 맞게 변환하기

- 게임 재시작시 세션에 남아있던 바둑판 위치 초기화

(4) 클라이언트-서버간 통신 연결 (url 매핑 및 ajax 통신)

- 오목 게임 접속 url 매핑

- ajax를 이용한 파라미터 전달

- 오목 여부에 따른 결과값 클라이언트에 응답

3. 소감

- 간단한 부분처럼 보여도 막상 생각대로 코드를 짜고 표현해내는 것이 쉽지 않음

- 서로 할 수있다고 믿어주고 기다려줘서 좋았다

- 스스로 해낼 수 있어서 성취감이 있었다

- 개인마다 생각하는 방향이 다르고, 코드에 녹여가는 방법이 다르다는 점을 느낌
