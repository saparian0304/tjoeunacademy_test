신고기능 - 게시판, 댓글
 - 부적절한 언행이나 내용이 있는 게시물 및 댓글을 신고할 수 있는 기능
 - 상세페이지에서 신고버튼 클릭 -> 신고사유 및 세부내용 제출 -> 관리자 검토 및 처리
 - 구현내용 : 
    - 신고시 신고게시물 내용을 참조할 수 있도록 팝업창으로 구현
    - (관리자)체크박스를 이용한 일괄처리 기능 구현
    - (관리자)해당 게시물의 누적신고수 확인
    - (관리자)컬럼별 정렬기능 및 상세 검색 기능 구현 : 신고구분, 신고일자, 처리구분, 처리일자, 신고자 및 피신고자 닉네임

책갈피기능 - 게시판
 - 사용자마다 원하는 게시물을 수집하고 챙겨볼 수 있는 책갈피 기능 구현
 - 상세페이지에서 아이콘 클릭으로 원하는 게시물 수집 가능
 - 핵심 구현 기능 : ajax
 - 수집된 책갈피 수 표현, 로그인 여부에 따라 아이콘 색상 변화

좋아요기능 - 게시판, 댓글
 - 게시물 또는 댓글에 좋아요 버튼을 눌러 호감을 표시하는 기능
 - 좋아요 수에 따라 화제 게시글 및 댓글을 확인할 수 있음
 - 핵심 구현 기능 : ajax
 - 받은 좋아요 수 표현, 로그인 여부에 따라 아이콘 색상 변화


채팅 - 개인 채팅, 오픈채팅, 강퇴기능
 - 커뮤니티 내 회원들과 실시간으로 소통할 수 있는 채팅기능 구현
 - 핵심 구현 기능 : 웹소켓
 - 친구채팅 : 친구인 회원과 1대1로 대화할 수 있는 서비스
    - 가장 최근에 대화를 나눈 친구가 목록 최상단에 오도록 구현
 - 오픈채팅 : 같은 관심사를 가진 회원끼리 모여서 대화할 수 있는 1대다 채팅 서비스
    - 오픈채팅 개설시 최대 참여인원 설정 가능
    - 방장 및 강퇴기능 구현
    - 참여중인 채팅방만 따로 볼 수 있도록 별도의 목록창 구현
    - 닫힌 채팅방은 자동적으로 목록에서 사라지도록 구현
    - 회원별로 채팅방 참여시점 이후에 대한 메시지만 확인할 수 있도록 구현

관리자게시판 - 게시글 관리 및 검색, 정렬 기능
 - 한 곳에서 모든 게시판의 게시물을 관리할 수 있도록 관리자 게시판 구현
 - 정렬 기능 : 게시판 헤더 선택시 해당 조건으로 정렬되도록 구현
 - 관리자 전용 게시물 작성기능 구현
 - 상세 조건 검색 기능 구현 : 제목, 내용, 닉네임 , 댓글내용, 기간지정 검색, 게시판 및 말머리 구분
